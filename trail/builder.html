📦 Standalone trail: a zip containing a folder of separate pages — index.html (the overview) and one HTML file per game, with images kept as their own files. Each page can be hosted behind its own link (or uploaded as a stqry screen); scores are shared across them via the stqry storage bridge.

In stqry: create a screen (or QR code) per game link — opening it drops the visitor straight into that game with the trail attached. The overview link is the final-score page; it also lists and links every game, so for an "any order" trail it can even be the only link you hand out. Scores are saved through the stqry storage bridge, so they survive navigation between screens in the app.
